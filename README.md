# DBFarma
trabalho desenvonvido na disciplina de Banco de Dados II

O nome do sistema é DBFarma e tem o intuido ser aplicado em alguma farmácia

Suas principais funcionalidades são: Cadastrar produtos, clientes, emitir relatórios entre outros.

Sistema Desenvolvido por Eduardo Ludwig, Elton Albugiere, Jonathan Dalpiaz e Maiky Ramon.
